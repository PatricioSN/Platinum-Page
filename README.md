# Platinum-Page
A simple webpage developed in HTML and CSS to practice structure, layout, styling, and small visual effects.

## Description
This project is an experiment focused on hands-on front-end practice.
The structure is clean: organized HTML and a separate CSS file to control layout, typography, spacing, and visual effects.


## Features
- Structured cards
- Image hover effects
- Custom button styling


## Technologies Used
- HTML
- CSS


## Purpose 
- HTML page structure
- CSS styling
- Semantics
